# oxytocin-bioprocess-design

Oxytocin Bioprocess Design — Recombinant Production in E. coli
UBC CHBE | Jan 2026 – Present
Overview
Designing a fed-batch fermentation process for recombinant oxytocin expression in E. coli as part of a UBC Chemical Engineering design project. Work includes growth medium specification, mass balances, equipment sizing, and process flow diagram development.
Current Progress

 Mass balances across upstream fermentation and downstream recovery
 Process flow diagram (PFD) — fermentation, cell lysis, chromatographic purification
 Bioreactor and key equipment sizing
 Literature review on recombinant peptide production in E. coli

Tools
Python, Excel, literature (peer-reviewed journals via UBC Library)
Status: In progress
